### Thành tựu Tuần 11:

* Học kiến trúc data lake concepts và best practices.
* Xây dựng data lake trên S3 với cấu trúc folder có tổ chức.
* Triển khai data lake security với IAM policies và Lake Formation.
* Crawl data sources sử dụng AWS Glue crawlers cho metadata discovery.
* Query S3 data sử dụng Amazon Athena với SQL.
* Phân tích AWS Cost and Usage Reports với Athena queries.
* Kết nối Amazon QuickSight với Athena data sources.
* Tạo interactive visualizations và dashboards trong QuickSight.
* Chia sẻ dashboards với stakeholders và cấu hình permissions.
* Tham gia Data Engineering Immersion Day workshops.
* Khám phá AWS AI services (Rekognition, Comprehend, Translate).
* Query VPC Flow Logs với Athena cho network analysis.
* Xây dựng comprehensive cost optimization dashboard trong QuickSight.

### Mục tiêu Tuần 9:

* Nắm vững thực hành DevOps với các dịch vụ CI/CD của AWS.
* Học Elastic Beanstalk cho triển khai ứng dụng.
* Xây dựng và triển khai hệ thống quản lý tài liệu (Dự án 2).

### Thành tựu Tuần 9:

* Tạo Git repositories trong CodeCommit và GitHub cho version control.
* Xây dựng automated CI/CD pipelines sử dụng AWS CodePipeline.
* Cấu hình CodeBuild cho automated application builds và tests.
* Triển khai Node.js applications lên Elastic Beanstalk.
* Cấu hình environment properties và quản lý application versions.
* Triển khai rolling updates cho zero-downtime deployments.
* Xây dựng CRUD API với Lambda cho quản lý tài liệu.
* Tích hợp AWS Amplify cho authentication và file storage.
* Kết nối frontend application với API Gateway endpoints.
* Thiết lập CloudFront distribution cho global content delivery.
* Triển khai search functionality sử dụng Amazon OpenSearch.
* Cấu hình end-to-end CI/CD pipeline cho dự án.
* Hoàn thành serverless web application workshop.
* Xây dựng real-time serverless chat application.

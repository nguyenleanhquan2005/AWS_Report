### Mục tiêu Tuần 5:

* Nắm vững Infrastructure as Code sử dụng CloudFormation và AWS CDK.

### Thành tựu Tuần 5:

* Viết thành công CloudFormation templates bằng YAML cho tự động hóa hạ tầng.
* Triển khai VPC stacks sử dụng CloudFormation với resource dependencies phù hợp.
* Xử lý stack updates và phát hiện configuration drift.
* Cài đặt và cấu hình AWS CDK cho infrastructure as code.
* Viết infrastructure code bằng Python/TypeScript sử dụng CDK constructs.
* Triển khai EC2 instances theo chương trình qua CDK.
* Khám phá advanced CDK constructs cho ECS và EKS.
* Triển khai static sites sử dụng AWS Amplify Console với CI/CD.
* Thiết kế giải pháp WordPress high availability trên AWS.
* Refactor networking infrastructure sử dụng CDK best practices.

* Nắm vững kiến trúc serverless với Lambda và API Gateway.

### Thành tựu Tuần 8:

* Nắm vững phát triển và triển khai AWS Lambda function.
* Tích hợp Lambda với API Gateway cho RESTful APIs.
* Nghiên cứu microservices architecture patterns và best practices.
* Khám phá event-driven architecture với các dịch vụ AWS.
* Thiết lập SQS queues cho xử lý message bất đồng bộ.
* Cấu hình SNS topics cho pub/sub messaging patterns.
* Thiết kế workflows phức tạp sử dụng AWS Step Functions.
* Triển khai Lambda triggers từ SQS cho event processing.
* Xây dựng serverless backend hoàn chỉnh với Lambda và DynamoDB.
* Tạo REST và HTTP APIs sử dụng API Gateway.
* Triển khai user authentication với Amazon Cognito.
* Tích hợp frontend application với serverless backend.
* Cấu hình custom domains và SSL certificates cho APIs.
* Test end-to-end application flow và error handling.
* Tự động hóa serverless deployment sử dụng AWS SAM.
* Tạo CI/CD pipeline cho serverless applications.
